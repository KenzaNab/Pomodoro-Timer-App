# Pomodoro Timer

Focus timer with Pomodoro technique, task list, audio notifications and auto mode switching.

> Vue 3 · Pinia · Vite · Web Audio API

## Features
- 25min Focus / 5min Short Break / 15min Long Break
- Auto-switch between modes
- Circular progress timer
- Audio beep on completion (Web Audio API)
- Task list with completion tracking
- Session counter (4 pomodoros = long break)
- Color themes per mode

## Quick start
```bash
npm install && npm run dev
```
